
# Crypto Space App [ Jetpack Compose ]

Follow crypto currency prices and latest crypto news in one app. Built with Kotlin, Jetpack Compose.



## Built By

**Language:** Kotlin

**Libs:** Live Data, View Model, Retrofit, Coil, WebView, Navigation, Hilt
